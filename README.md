# NEXT AURA COMMUNICATIONS Website

A premium, modern, and fully responsive corporate website for NEXT AURA COMMUNICATIONS. 

## Tech Stack
* **Angular 20** (Standalone Components, Signals, Lazy Loading)
* **TypeScript**
* **Tailwind CSS** (v3, styling via custom variables and utilities)
* **GSAP** (Smooth scroll and micro-animations)

## Key Features
* High-performance, fast-loading architecture
* Modern "Glassmorphism" and Neon-accented Dark Mode aesthetic
* Dedicated SEO meta management (JSON-LD, Dynamic Meta tags)
* Integrated Instagram Reels carousel above footer
* Accessible forms, buttons, and layouts

## Development Server

Run `npm run dev` or `npm start` for a dev server. Navigate to `http://localhost:4200/`. The application will automatically reload if you change any of the source files.

## Build

Run `npm run build` to build the project. The build artifacts will be stored in the `dist/` directory.

## Search Engine Optimization (SEO) setup
1. **Google Search Console**: Edit `src/index.html` and replace `YOUR_VERIFICATION_CODE_HERE` with your actual Google Verification Tag.
2. **Sitemap**: Check `public/sitemap.xml`
3. **Robots**: Check `public/robots.txt`
